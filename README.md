Personal Clawdbot agent workspace.

A digital familiar for your digital life.

## Setup

```bash
clawdbot gateway start
```

## Workspace Structure

- `AGENTS.md` - Agent configuration and rules
- `SOUL.md` - Agent personality and core truths
- `USER.md` - Information about your human
- `TOOLS.md` - Environment-specific tool notes
- `HEARTBEAT.md` - Periodic heartbeat tasks
- `IDENTITY.md` - Agent identity
- `canvas/` - Canvas UI assets
